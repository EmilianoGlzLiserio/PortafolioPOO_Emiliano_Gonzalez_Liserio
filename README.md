📖 Descripción General
Este repositorio contiene una colección de proyectos desarrollados durante el segundo semestre de la carrera para la materia de Programación Orientada a Objetos. 
Los ejercicios y sistemas integradores demuestran la aplicación práctica de conceptos fundamentales de programación, desde el manejo de flujos básicos hasta la implementación avanzada de clases, encapsulamiento, herencia y polimorfismo.

🚀 Proyectos Incluidos
1. Sistema de Usuarios con Roles (Python)
Directorio: Act_evaluacion/

Descripción: Un sistema de gestión por consola que implementa los cuatro pilares de la POO. Define una clase base Usuario de la cual heredan clases específicas como Admin, Cliente e Invitado.

Conceptos Clave: Herencia, Polimorfismo, Sobrescritura de métodos (acceso_sistema()), validación de datos con expresiones regulares (RegEx) y manejo estructurado de excepciones (ValueError).

2. Monitor Energético para DataCenter (PHP)
Directorio: monitor_energetico/

Descripción: Una aplicación web que calcula el consumo de energía total simulado de un centro de datos utilizando métodos de integración numérica.

Conceptos Clave: Creación de la clase IntegradorNumerico, uso de namespaces, manejo de excepciones (try-catch) para validación de datos de entrada y conversión matemática de unidades (Joules a kWh).

3. Modelo de Logística de Paquetes (PHP)
Directorio: src/logistica/

Descripción: Modelado en código de un sistema básico de logística.

Conceptos Clave: Instanciación de objetos, definición de propiedades públicas y privadas para demostrar el encapsulamiento, e interacción entre múltiples clases independientes (Paquete y Sensor).

4. Sistema de Inventario (PHP)
Directorio: Act_integradora/

Descripción: Interfaz dinámica para la captura y registro de productos de un inventario. Procesa los datos capturados para calcular métricas de negocio.

Conceptos Clave: Manejo de arreglos paralelos, captura de datos por método HTTP POST, funciones nativas de arrays y estructuración de tablas dinámicas en HTML.

5. Bitácora de Actividades (PHP)
Directorio: Act_integradora_u6/

Descripción: Un sistema de registro de actividades que guarda información de manera persistente.

Conceptos Clave: Manipulación de archivos de texto plano (file_put_contents en modo FILE_APPEND), manejo de bloqueos de archivo (LOCK_EX) y patrón de redirección seguro (PRG - Post/Redirect/Get) para evitar
la duplicación de envíos de formulario.

🛠️ Tecnologías y Herramientas
Lenguajes: Python 3.x, PHP 8+

Frontend: HTML5, CSS3

Paradigmas: Programación Orientada a Objetos (POO), Programación Procedural Estructurada.
